직접 입력 안내

3개 시트 영역을 나눠 입력합니다. Room·장비·전실은 행을 추가해서 작성하고, 입력값으로 XLSX를 생성해 같은 백엔드 파이프라인으로 실행합니다.

실행 옵션
아래 옵션은 실행 시 백엔드 `/runs` 요청에 같이 전달됩니다.
전실 중복 제거 (exclude_airlock_rooms)
rooms[]에서 전실 Room 제외 (48→30). airlocks·adjacency 불변.
생물안전 격리 강제 (bio isolation)
전실 흐름 타입을 sink/bubble로 강제 (cascade→sink).
RAG 검증 실행 (run_validation)
끄면 Rule Engine 결과만 빠르게 산출합니다.

아직 결과가 없습니다. 위에서 URS를 업로드하고 파이프라인 실행을 눌러주세요.
4DRAWING

도면 (Drawing Agent)

7블록 사양을 좌표로 배치해 실제 GMP layout SVG를 생성합니다. 미리보기와 다운로드는 백엔드 `/drawing` 엔드포인트에서 받아옵니다.

Drawing Agent — 확인 중

입력 계약: 7블록 JSON · 출력 포맷: SVG · backend.app `/runs/{id}/drawing`

variant 3개를 생성하면 여기서 도면을 전환하고, 원하는 동선만 켜서 볼 수 있습니다.

Rule Engine · Validation Agent 로직 확인

요청 시 적용된 룰의 근거와 검증 판정(verdict)을 열어볼 수 있습니다. 모든 결정은 추적 가능(rationale)합니다.
