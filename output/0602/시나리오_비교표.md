# 발표 도면 5종 — URS 입력 → 내부 처리 → 출력 차이 정리

> **한 문장 요약**: *같은 엔진·같은 배치 알고리즘*에 **URS 입력(공정 규모·품목 수·무균충전 여부)** 만 바꾸면,
> 룰엔진이 방 구성(개수·등급·에어록)을 다르게 산출하고 → Drawing Agent 가 GMP 청정도 구배(NC→D→C)
> 토폴로지로 자동 배치 → **도면이 그 입력에 맞게 달라진다.**

파이프라인:

```
URS 입력 ──▶ Rule Engine ──▶ 7-block 출력 ──▶ tier1 어댑터 ──▶ Drawing Agent ──▶ SVG 도면
(공정 결정)   (15개 GMP 룰)   (rooms/AL/      (필드 정규화)    (NC→D→C 구배 솔버)
                            adjacency/flow)
```

## ③ 시나리오별 상세 — "URS → 엔진 → 도면" 흐름

### 기준 `v2_soyeon` (mAb 8000L)
- **URS**: 단일클론항체(mAb), 배양 8000L, 단일 품목, 무균충전 미포함.
- **엔진**: 표준 공정 train (Media→Buffer→Inoculation→Cell Culture→Harvest→Purification 1→2) + 준비/세척 + 지원(저장·IPC) + NC(사무·화장실·CIP·모니터링). 전부 Grade C/D.
- **도면**: NC→Grade D→[D 세로복도]→Grade C 공정. 공정행 상(Media~Harvest 5) / 하(Purif 1·2·Prep·Wash 4), 중앙 supply + 양측 return + 가운 게이트.

### 소규모 파일럿 `small_pilot_2000L`
- **URS 노브**: 배양 규모 ↓ (8000→**2000L**), 정제 2단계 **미사용**, 지원·NC 최소 구성.
- **엔진 효과**: 방 수 ↓(48→34), 정제 suite 1개, 공정 면적 ×0.55, 에어록 ↓(18→14).
- **도면**: 전체 축소(캔버스 89×48→78×43m), 정제실 1개, 지원/NC 구역 얇아짐. → **"규모를 줄이면 시설도 콤팩트해진다"**.

### 대규모 멀티프로덕트 `large_multiproduct`
- **URS 노브**: **다품목 병렬 생산** + QC 시험실 온사이트 + 원료/완제 창고 + 유틸리티 확장.
- **엔진 효과**: **병렬 공정 라인 B** 추가(Cell Culture B·Harvest B·Purification B) + QC Lab·Micro Lab + 창고 3 + 유틸 NC 4 → 방 48→60, AL 18→22.
- **도면**: 캔버스 최대(109×59m), 공정행이 상6/하6으로 늘고 **하행에 라인 B**가 자리. D·NC 구역 대폭 확장. → **"품목·시험 늘면 병렬 라인과 지원구역이 붙는다"**.

### 무균충전 온사이트 `aseptic_filling`
- **URS 노브**: **aseptic_filling_onsite = True** (원액→완제 무균충전을 같은 건물에서).
- **엔진 효과**: 베이스(전부 C/D)에 **Grade A 무균충전실 + Grade B 충전배경/가운** + Capping·Lyophilizer 추가 → 등급에 **A·B 첫 등장**, 에어록 18→22(더 엄격한 cascade).
- **도면**: 하행에 **Grade A(대각 해치) + Grade B(에메랄드)** 충전 suite가 시각적으로 명확히 출현. → **"무균충전을 켜면 최고등급 구역이 도면에 새로 나타난다"** (입력→출력 차이의 가장 강한 시각 증거).

### 중앙블록형 `perimeter_ring`
- **URS**: 기준과 **동일 입력**.
- **차이**: Drawing Agent의 **배치 전략만 교체**(strip-band 구배 → 중앙 공정 그리드 + ㅁ자 외곽 갤러리).
- **의미**: 같은 방 구성도 **여러 토폴로지로 배치 가능** — 엔진(무엇을)과 배치(어디에)의 책임 분리를 보여줌.

## ④ 공통 엔진·구현 (5종 모두 동일하게 적용)

GMP 청정도 구배 토폴로지 — 사용자(GMP 전문가) 검수 피드백 6건 반영:

- **가로 청정도 구배 NC → Grade D → Grade C** (교차오염 차단, 인접은 한 등급차)
- **Grade D 세로 복도** = 인원/자재 진입 + return 배출 공용 통로
- **가운 게이트(C)** = D복도 ↔ supply 복도 접점 입실 게이트 (EU GMP Annex 1)
- **중앙 supply → 공정행 통과 → 양측 return → D복도 배출** = one-way flow
- 양측 return 복도 실제 배치 (정제실 포함 전 공정행이 return 인접)
- 작은 방 종횡비 클램프(슬리버 방지) · 건축 표준 여닫이 도어 · 차압 기반 개폐방향
- 동선 4유형 색·점선: 인원(indigo)/자재(teal)/제품(violet)/폐기물(rose)
